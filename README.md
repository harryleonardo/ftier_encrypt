# Ftier Encrypt
Ftier Encrypt is a repository that serve encryption for incoming request.

## Usage
1. Manual run
    ```shell
        make start
    ```
2. Docker run
    ```shell
        docker-compose up
    ```

### Related Library
1. Echo Labstack as a Framework
2. Gorequest for HTTP Call
3. Go mod for managing depedencies
4. Viper for managing the configuration
